# PrepXL Full-Stack Assignment

This repository contains the implementation of the PrepXL pre-interview assignment.

## Projects Included

### 1. Circular Audio Visualizer
- Built using Web Audio API and Canvas
- Real-time circular frequency visualization
- Smooth animations with live microphone input
- Implemented using Vite + JavaScript

### 2. Real-Time Streaming Transcription
- Frontend captures microphone audio using MediaRecorder API
- Backend uses WebSockets for real-time communication
- Demonstrates low-latency audio streaming and partial transcription output
- Architecture is designed to be extensible for future STT integrations

## Tech Stack
- HTML, CSS, JavaScript
- Web Audio API
- Canvas API
- Node.js
- WebSockets

## Notes
- Gemini API was not directly used due to lack of native low-latency audio streaming STT support
- Focus of the assignment is on real-time streaming architecture and system design
